BorrowIt – Peer-to-Peer Lending Platform

BorrowIt is an innovative web platform that promotes the sharing economy by enabling users to easily lend and borrow everyday items. It helps reduce unnecessary purchases and supports eco-responsible behavior.

Users can browse items by category, manage rentals using built-in calendars and digital contracts, and securely exchange items within a trusted community.

A key feature is the integration of AI-powered facial recognition during user registration, ensuring secure and automated identity verification to build trust among users.

The platform is built with Angular for the frontend and Spring Boot for the backend. It uses JWT for secure authentication, MySQL for database storage, and is styled with Bootstrap. The project is deployable both locally and on Azure.

This project contributes to several UN Sustainable Development Goals (SDGs):

  • SDG 9 – Industry, Innovation, and Infrastructure
  • SDG 11 – Sustainable Cities and Communities
  • SDG 12 – Responsible Consumption and Production

BorrowIt is developed by a student-driven team committed to sustainable, ethical, and impactful technology.
