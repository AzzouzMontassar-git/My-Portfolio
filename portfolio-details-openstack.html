Private Cloud Infrastructure using OpenStack

This project involved the design and deployment of a complete private cloud infrastructure using OpenStack. Core components such as Keystone (identity), Nova (compute), Neutron (networking), Glance (image service), and Horizon (dashboard) were successfully deployed.

The environment was built on Ubuntu 22.04 using KVM virtualization in a multi-node architecture. We used Heat to automate resource provisioning and Ansible to manage configuration and service deployment across nodes.

On top of this infrastructure, a Kubernetes cluster was launched using the instances provisioned in OpenStack. This enabled efficient orchestration and scaling of containerized applications.

For observability, we integrated Prometheus for monitoring and Grafana for data visualization, ensuring real-time insights into system and service health.

In parallel, a MySQL database was deployed on Azure using AKS (Azure Kubernetes Service), achieving a hybrid cloud architecture that combines on-premise (OpenStack) and public cloud (Azure) environments.

This project provided extensive hands-on experience with cloud computing, infrastructure as code, hybrid cloud integration, and modern DevOps tools.
