E-learning Web Application

E-learning Platform is a comprehensive online education solution developed during my internship at "Les Jeunes Codeurs" association. The application provides an interactive learning environment with separate interfaces for students and administrators.

The platform was designed to address the lack of specialized e-learning solutions in our local market. While global platforms like Coursera and Udemy exist, we aimed to create a tailored solution for our regional needs, offering a user-friendly interface for students to discover and enroll in courses, while providing administrators with robust management tools.

Key Features

  • Student Portal: Browse available courses, filter by category/level/dates, enroll online, and access course content
  • Admin Dashboard: Manage user accounts, courses, chapters, and monitor student enrollments
  • Course Management: Add/edit courses and organize content into chapters
  • User Management: Handle student registrations and profiles

Technical Implementation

The application follows a 3-tier architecture with clear separation between presentation, business logic, and data layers:

  • Frontend: Developed with Angular 14 using the MVVM pattern for clean separation of concerns
  • Backend: Built with Spring Boot following a layered architecture (Web, Service, Repository, Model)
  • Database: MySQL for data persistence

Methodology

The project was developed using Agile (Scrum) methodology, with regular sprints and stakeholder feedback incorporated throughout the development process. This approach ensured we could adapt to changing requirements while maintaining project timelines.
